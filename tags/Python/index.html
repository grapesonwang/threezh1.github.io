2. __import__()

单独使用__import__() 可以直接加载模块，但是当需要动态加载类、函数时，就需要配合getattr来实现。

实现步骤：

  1. 获取模块名(module_name)
  2. 使用__import__(module_name)导入Python模块
  3. 使用getattr(module_name, class_name/function_name)获取类、方法的对象
