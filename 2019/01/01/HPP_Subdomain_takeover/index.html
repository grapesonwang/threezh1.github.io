HPP

测试：

存在于POST,GET之中。

步骤：

  1. page?par1=vall
  2. page?par1=HPP_TEST1
  3. page?par1=val1&par1=HPP_TEST1

更详细的：https://www.acunetix.com/blog/whitepaper-http-parameter-pollution/

Use to:

  1. 取代现有的编码HTTP参数。
  2. 更改或修改预期/正常应用程序行为。
  3. 访问并利用未正确控制的变量。
  4. 绕过WAF规则或输入验证机制。
